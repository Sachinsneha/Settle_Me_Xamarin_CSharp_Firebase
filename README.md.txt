# Settle Me – Xamarin App for International Students

**Settle Me** is a mobile app built with **Xamarin (C#)** and powered by **Firebase**, designed specifically to help international students manage everything they need upon arriving in a new country. From finding a room to booking rides, connecting with local agents, managing essential tasks, and chatting with others — this all-in-one app simplifies the transition process.

---

## 🎯 Purpose

Many students studying abroad face difficulty navigating their new environment. Settle Me is created to support them with practical features like:

- 🏠 Finding rental rooms
- 🚗 Booking local rides
- 👷 Hiring service agents (e.g., movers, setup help)
- 📝 Managing arrival tasks
- 💬 Real-time communication

---

## 🚀 Features

### ✅ Authentication
- Firebase email/password login
- Firebase email verification
- Persistent user session

### 🏠 Rental System
- Browse rental listings (rooms/apartments)
- View images, pricing, availability
- Contact landlords/owners directly

### 🚗 Ride Booking
- Offer or request rides
- View booking status
- History of past rides

### 👷 Agent Booking
- Browse verified agents (e.g., movers, setup help)
- View availability and request services
- Booking confirmation & management

### 📝 Task List (Arrival Checklist)
- Personalized checklist for newly arrived students
- Mark tasks as completed
- Tasks include: get SIN, open bank account, get transit card, etc.

### 💬 Messaging
- Real-time 1-to-1 chat with other users
- Firebase Firestore backend
- Online/offline status indicator

---

## 🧑‍💻 Tech Stack

- **Language**: C#
- **Framework**: Xamarin.Forms (Cross-platform)
- **Backend**: Firebase (Auth, Firestore, Storage, Cloud Messaging)
- **Architecture**: MVVM (Model-View-ViewModel)
- **IDE**: Visual Studio 2022+
